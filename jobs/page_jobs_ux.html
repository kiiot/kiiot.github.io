公司簡介

愛物聯成立於2014年2月，是致力於提供零售、餐飲、旅館和商場一站式O2O到CRM解決方案的公司。從無線網路、Beacon、QRCode及數位看板應用到數位行銷、數據收集、會員經營與再行銷完整軟硬整合服務方案，豐富的雲端後台行銷應用及客戶關係管理，幫助客戶降低行銷成本，解決經營的盲點與痛點，提供最佳的銷售策略分析。

市場趨勢

在BYOD（Bring Your Own Device）的趨勢下，店面內的網路建置是被列為必須的費用，店內的無線網路被視為剛性需求也是店面競爭力。另一方面，電子商務的出現也改變零售業的經營模式，從單一目的性的購物方式，轉為重視休閒性，零售業現在已經步入Wi-Fi 2.0：Wi-Fi as a Service的時代，必須要從費用支出的思維轉變成投資。

以無線創造無限商機

愛物聯重新定義WiFi，透過物聯技術取得入口數據，藉由數據服務做好智慧行銷，分析數據形成商業智能，智慧零售解決方案。

工作說明

愛物聯的WiFi不只是上網工具，更是絕佳的活動行銷與社群互動管道，如何將過客變成忠誠粉絲，絕佳的用戶體驗是最為重要的。愛物聯UI/UX設計需要了解站在用戶角度，思考如何設計視覺與互動流程，並追蹤與分析用戶行為，持續優化產品。

工作內容

專注於產品帶給用戶的感受，透過設計改善創造出最棒的使用者體驗。

  • 參與產品設計與討論，規劃產品介面與使用流程
  • 準備Flow Chart / UI Flow 協同團隊開發，使用Wireframes或Prototype 簡易原型有效率地溝通想法，讓團隊明白設計理念與呈現效果。
  • 與前後端工程師協作，開發產品視覺與互動設計
  • 優化使用者體驗。

條件要求

誠摯邀請擁有良好程式能力與學習熱忱的您來與我們一同成長打拼。

基本條件：

  • 熟悉 HTML, CSS, Javascript
  • 熟悉 Responsive Web Design (響應式網頁設計) for 跨平台
  • 在wireframes, user flows, process flows 有經驗
  • 出色的視覺設計能力，對用戶操作行為好奇並有實驗精神
  • 履歷請附上作品集簡介或作品連結

加分技能：

  • 熟悉 Ajax 與 jQuery
  • 熟悉 Bower、Gulp 或 Grunt 前端管理工具
  • 熟悉 Git 和 linux 等開發環境
